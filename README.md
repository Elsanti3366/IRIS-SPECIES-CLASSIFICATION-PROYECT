# Iris Species Classification Dashboard

## Overview
End-to-end machine learning project that classifies Iris flower species using Random Forest. Built with Streamlit and deployed on Streamlit Cloud.

## Live Dashboard
[View on Streamlit Cloud](#) ← Replace with your deployed URL

## Features
- Model metrics: Accuracy, Precision, Recall, F1-Score
- Confusion matrix and feature importance
- Interactive prediction panel with sliders
- 3D scatter plot showing new sample vs dataset
- Histograms, scatter matrix and correlation heatmap

## Workflow
1. Data Understanding — Iris dataset (150 samples, 4 features, 3 classes)
2. Preprocessing — StandardScaler normalization
3. Train/Test Split — 80/20 stratified
4. Model — Random Forest (100 trees)
5. Evaluation — Accuracy, Precision, Recall, F1, Confusion Matrix
6. Deployment — Streamlit Cloud

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run Proyect.py
```
